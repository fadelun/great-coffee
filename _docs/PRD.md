# Role

Act as a Senior Frontend Developer and UI/UX Designer specialized in React ecosystem.

# Project Goal

Create a production-ready, high-performance Coffee Shop Website Template using **Next.js (App Router)**, **TypeScript**, and **Tailwind CSS**.

# Tech Stack Requirements

- Framework: Next.js (App Router)
- Styling: shadc ui (Mobile-first approach)
- Icons: Lucide React
- Animations: Framer Motion (subtle fade-ins)
- Fonts: Google Fonts (Lato for body, Abril Fatface for headings)
- Images: Use placeholder images from Unsplash (source: coffee shop)

# Design System

- Theme: Warm, Earthy tones (Coffee Brown, Cream, Dark Charcoal).
- Vibe: Modern, Artisan, Cozy, Minimalist.
- Layout: Clean whitespace, grid-based menu, sticky navbar.

# Core Features & Pages to Build

1. **Global Components:**
   - Responsive Navbar (with mobile hamburger menu).
   - Footer (Social links, Newsletter, Quick links).
2. **Home Page:**
   - Hero Section (Headline, Subheadline, CTA "Order Now").
   - Featured Menu (Grid of 3 best sellers).
   - About Snippet (Short story + Image).
   - Testimonials (Carousel or Grid).
3. **Menu Page:**
   - Category Filter (All, Coffee, Non-Coffee, Food).
   - Menu Items Card (Image, Name, Description, Price, "Add" button).
4. **Contact/Location Page:**
   - Google Maps Placeholder.
   - Opening Hours Table.
   - WhatsApp Float Button.

# Constraints & Best Practices

- Use Functional Components with Hooks.
- Ensure strict TypeScript typing.
- Optimize images using Next.js `<Image />` component.
- Ensure accessibility (ARIA labels, semantic HTML).
- Code must be modular (split components into `/components` folder).
- Do not use external CSS files, use Tailwind utility classes.

# Output Instructions

1. First, outline the **project folder structure**.
2. Provide the code for **`tailwind.config.ts`** with the custom color theme.
3. Provide the code for the **`layout.tsx`** (including fonts).
4. Provide the code for the **`page.tsx` (Home)** with all sections.
5. Provide the code for the **`components/Navbar.tsx`** and **`components/MenuCard.tsx`**.
6. Explain how to run the project locally.

# Tone

Professional, Clean, Efficient Code.
